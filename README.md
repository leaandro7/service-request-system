Sistema de Chamados - CallTech

Este projeto foi desenvolvido como parte da segunda avaliação da disciplina CAW. O objetivo do trabalho foi criar um sistema de gerenciamento de chamados para a empresa CallTech, utilizando PHP para implementação de um CRUD (Create, Read, Update, Delete).
O sistema permite que usuários registrem novos chamados, visualizem chamados existentes, atualizem informações e excluam registros quando necessário. Com isso, a CallTech pode gerenciar solicitações de suporte de forma mais organizada e eficiente.

**Funcionalidades:**
Create: Permite cadastrar novos chamados.
Read: Permite consultar e listar chamados existentes.
Update: Permite atualizar informações de chamados já cadastrados.
Delete: Permite remover chamados do sistema.

**Tecnologias Utilizadas:**
PHP: Linguagem principal para implementação do sistema.
MySQL: Banco de dados para armazenamento dos chamados.
HTML/CSS/JS: Interface básica para interação do usuário.
